We aren't building a "soft" SaaS landing page with gradients and rounded corners. We are building a technical manual for a weapon system.

Here is a breakdown of the Visual Theme & Styling Strategy for tx-2.dev, leaning into the "TX-2 / Sutherland / F-22" vibe.

1. The Core Aesthetic: "Phosphor & Void"
Concept: The original MIT TX-2 computer used vector displays with glowing green phosphor on pitch-black screens. We modernize this.

Background: #050505 (Almost pure black, "Void").

Primary Accent: #00FF41 (The classic "Terminal Green") or a slightly more modern #39FF14 (Neon Green).

Secondary Accent: #FF3333 (Alert Red - for the "Thundering Herd" warnings).

Text: #E0E0E0 (Off-white, high contrast but not eye-bleeding).

CSS Variable Idea:

CSS

:root {
  --color-void: #09090b; /* Zinc-950 */
  --color-phosphor: #4ade80; /* Green-400 */
  --color-grid: #27272a; /* Zinc-800 */
  --font-mono: 'JetBrains Mono', monospace;
  --font-display: 'Space Grotesk', sans-serif;
}
2. Typography: "The Engineering Manual"
The font choice is critical. It needs to feel like a blueprint.

Headings (H1-H3): Space Grotesk or Chakra Petch. Something blocky, wide, and technical.

Body Text: Inter or Geist Sans (Vercel's font). Clean, readable, neutral.

Code Blocks: JetBrains Mono or Fira Code. Ligatures are mandatory.

3. UI Elements: "The HUD"
Avoid "cards" with drop shadows. Use Outlines and Grids.

Grid Lines: Use a subtle background grid pattern (like a CAD drawing or radar screen) that fades in and out.

Borders: 1px solid borders with sharp corners (border-radius: 0px or 2px). No 16px rounded corners.

Active States: When you hover a button, don't just change color. Invert it (Black text on Green bg). Or add "corner brackets" that appear.

5. Content Layout (The "Documentation")
Since this is a framework, the docs need to be pristine.

Sidebar: Fixed left. High density.

Right Column: "On this page" table of contents.

Code Snippets: The most important part.

Use Shiki or Prism for syntax highlighting.

Theme: Github Dark Dimmed or a custom "Phosphor" theme.

Feature: "Copy" button that says COPIED in green text.
